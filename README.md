# Carasco Ableton Extensions

Public distributable Ableton Live extensions.

This repository currently contains packaged `.ablx` extension archives. Each
extension folder includes its release package and its own README with usage
notes.

## Extensions

| Extension | Description | Package                                             |
| --- | --- |-----------------------------------------------------|
| [KeyPulse](KeyPulse/README.md) | Detects key and BPM for Ableton audio clips and tags the clip name. | [Download](KeyPulse/) |
| [SplitScope](SplitScope/README.md) | True mid-side extraction with a live spectrogram editor. | [Download v0.1.1](SplitScope/SplitScope-0.1.1.ablx) |

## Requirements

- Ableton Live with Extensions support.
- Extensions enabled in Live.

The Ableton Extensions SDK is still beta, so installation and extension-host
behaviour may change between Live/SDK releases.

## Installing

Download the `.ablx` package for the extension you want and install it through
Ableton Live's Extensions workflow.
